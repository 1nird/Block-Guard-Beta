Help & Guide

Everything you need to master Block Guard and protect your focus.

Getting Started

add_circle

1. Add Sites to Block

Go to the Block List tab and type any website domain (e.g. facebook.com). Hit Add. These sites will be blocked whenever a focus session is active.

bolt

2. Start a Focus Session

Click Start Focus Session in the sidebar or use the Focus Mode tab. Pick a focus level (Light Study, Deep Work, or Zen Mode) and start the timer.

calendar_month

3. Schedule Recurring Blocks

Go to Schedule and create recurring focus blocks. Pick days of the week, set times, and choose which sites to block for each block.

bar_chart

4. Track Your Progress

Open Insights to view focus trends, blocked attempt breakdowns by category, interception logs, and your overall focus score.

Focus Modes Explained

menu_book

Light Study — 25 min

Blocks only the most addictive social media sites (Twitter, TikTok, Instagram, Reddit). Good for reading, reviewing notes, and light research where you still need some sites available.

psychology

Deep Work — 50 min

A strict blocking mode that covers social media, streaming, news, and entertainment. Use this for focused coding, writing, studying, or any creative work that requires uninterrupted attention.

spa

Zen Mode — 90 min

The most aggressive mode. Blocks social media, video, news, shopping, gaming, and general entertainment. Designed for long deep-work sprints, exam prep, or high-stakes creative sessions.

Key Features

verified_user

Whitelist Mode

Toggle Whitelist Mode in Block List to reverse the blocking logic — instead of blocking listed sites, only listed sites will be allowed. Everything else gets blocked.

music_note

Ambient Soundscapes

The Focus Mode tab includes built-in ambient sounds (Rain, Forest, Brown Noise, White Noise) to help you maintain concentration while sessions are running.

download

CSV Export

Download your full interception log as a CSV from the Insights tab. Track patterns, share with accountability partners, or use the data for personal analytics.

schedule

Focus Scheduling

Schedule blocks have three site modes: Global (use your main block list), Custom (block specific sites only), or Allow Only (whitelist mode just for that block).

Frequently Asked Questions

What happens when I exit a focus session early?expand_more
Your elapsed focus time is still saved and added to your total. Blocking rules are removed immediately so you can browse normally again.
Can I block sites permanently without a timer?expand_more
Yes! In the popup, set the duration to 0 (shown as ∞). This will block sites indefinitely until you manually exit the session.
Does the schedule work when Firefox/Chrome is closed?expand_more
The schedule checks every minute while the browser is open. If the browser is closed during a scheduled block, it will activate as soon as you reopen it.
What's the difference between "custom sites" and "global list"?expand_more
Global list uses the main Block List you've set up. Custom sites let you define a separate set of sites per schedule block (or per focus level), independent of your main list.
Why does Google Search still work in whitelist mode?expand_more
Google Search is always allowed in whitelist mode so you can still look things up. This is intentional — blocking search would make the browser unusable.
Is my data synced anywhere?expand_more
No. Everything is stored locally in your browser's extension storage. Nothing is sent to any server. Your data is 100% private and never leaves your device.
How do focus level presets work?expand_more
Each focus level (Light Study, Deep Work, Zen Mode) comes with a preset list of commonly blocked distraction sites. These are applied automatically when you start a session with that level selected. You can customize the sites per level.
